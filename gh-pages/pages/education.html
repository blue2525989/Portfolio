Launch Code boot camp

I have most recently completed Lanuch Code's LC101 six month coding boot camp. The program is broken down into three units, an introduction to basic skills, next an introduction to building applications for the web and the MVC concept, and lastly a skill track is chosen and then we build web applications in the chosen language.

The first unit is taught in Python and features an introduction to basic programming concepts. Throughout the unit we build simple applications that teach the concepts of collections and object oriented design.

Next we dove into the concepts of coding for the web with building applications in Python with the google app engine as a container/server. This is where we started to learn the concepts of Model View and Control design. This was great because I was learning Spring in my free time and was able to take this home and apply it to my personal project.

Lastly we were given the option of a specific skill track to choose, either Java or C#, I chose Java because I already had four months of self learning invested in it. We were introduced to the Spring Boot framework as well as Thymeleaf for rendering the views and MySQL to persist all of the data.
